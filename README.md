# 电影订票管理系统9



## 1、项目介绍

电影订票管理系统9拥有两种角色

管理员：用户管理、电影类型管理、电影管理、订单管理、留言管理

用户：登录注册、购票、订单管理、查询等


## 2、项目技术

后端框架： Servlet

前端技术：jsp、css、JavaScript、JQuery

## 3、开发环境

- JAVA版本：JDK1.8（最佳）
- IDE类型：IDEA、Eclipse都可运行
- tomcat版本：Tomcat 7-10版本均可
- 数据库类型：MySql（5.7、8.x版本都可） 
- maven项目：否
- 硬件环境：Windows
